# SD-WAN Multi-Region Fabric (Cisco Viptela)

## Overview
This project demonstrates the design and implementation of a Cisco SD-WAN
Multi-Region Fabric for a large-scale enterprise network.

## Key Features
- Multi-Region Control Plane
- Dual Data Centers
- Dual Transport (Internet + MPLS)
- Centralized and Localized Policies
- High Availability and Failure Domain Isolation

## Target Audience
- Network Engineers
- SD-WAN Designers
- CCNP/CCIE Enterprise Candidates
